# Sorting

| Algorithm | Worst-Case       | Space    |
|-----------|------------------|----------|
| Insertion | O(n<sup>2</sup>) | O(1)     |
| Mergesort | O(n lg n)        | O(n)     |
| Heapsort  | O(n lg n)        | O(1)     |
| Quicksort | O(n<sup>2</sup>) | O(lg n)  |
| Counting  | O(n + k)         | O(k)     |
| Radix     | O(nk)            | O(n + k) |
| Bucket    | O(n<sup>2</sup>) | O(n)     |
